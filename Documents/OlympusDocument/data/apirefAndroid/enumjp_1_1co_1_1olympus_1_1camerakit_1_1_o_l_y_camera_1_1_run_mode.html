Member Data Documentation

Maintenance

Maintenance mode.

This mode does not work and is reserved for future expansion.

Playback

Playback mode.

This mode is used to view the captured image in the camera.

Playmaintenance

Play-maintenance mode.

This mode does not work and is reserved for future expansion.

Recording

Recording mode.

This mode is used to capture still images and record videos by the shutter via wireless communication.

Standalone

Standalone mode.

The run mode is this value immediately after connected to the camera.

If application changes to this mode from any other mode, several camera properties return to their initial states. For more information please refer to documentation of the camera list of properties.

Unknown

Mode where SDK does not work.

Run mode is sometimes set to this value when application is not connected or mode change is abnormally terminated.
